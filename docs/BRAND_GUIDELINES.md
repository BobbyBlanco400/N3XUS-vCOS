# N3XUS v-COS Brand Guidelines

## Overview
This document provides comprehensive guidelines for using the N3XUS v-COS logo and maintaining brand consistency across all platforms and media.

## Logo Philosophy
The N3XUS v-COS logo represents the world's first Virtual Creative Operating System, combining:
- **Hexagonal geometry**: Symbolizing connectivity, structure, and the networking of creative ideas
- **Gradient colors**: Representing the spectrum of creativity and innovation
- **Modern typography**: Conveying technological advancement and forward-thinking design
- **Glowing effects**: Suggesting digital energy and virtual presence

## Brand Colors

### Primary Palette
- **Cyan**: `#00d9ff` (RGB: 0, 217, 255)
  - Usage: Primary accent, highlights, interactive elements
  - Represents: Innovation, clarity, digital presence
  
- **Purple**: `#7b2ff7` (RGB: 123, 47, 247)
  - Usage: Secondary accent, gradients, creative elements
  - Represents: Creativity, imagination, artistic vision
  
- **Magenta**: `#ff006e` (RGB: 255, 0, 110)
  - Usage: Tertiary accent, emphasis, calls-to-action
  - Represents: Energy, passion, bold innovation

### Background Colors
- **Dark Navy**: `#0a0e27` (RGB: 10, 14, 39)
  - Usage: Primary backgrounds, dark mode
  
- **Deep Blue**: `#1a1f3a` (RGB: 26, 31, 58)
  - Usage: Secondary backgrounds, cards, panels

### Neutral Colors
- **White**: `#ffffff` - Text on dark backgrounds
- **Light Gray**: `#e0e0e0` - Secondary text
- **Dark Gray**: `#333333` - Text on light backgrounds

## Typography

### Primary Font
**Courier New** (Monospace)
- Usage: Main logo text, technical content, code examples
- Weights: Bold (700) for "N3XUS", Light (300) for "v-COS"
- Character: Technical, precise, digital

### Secondary Font
**Arial / Helvetica** (Sans-serif)
- Usage: Body text, descriptions, taglines
- Weights: Regular (400), Light (300)
- Character: Clean, readable, professional

## Logo Variations

### 1. Primary Logo (`n3xus-vcos-primary.svg`)
- **Dimensions**: 800x200px
- **Usage**: Website headers, presentations, marketing materials
- **Features**: Full logo with tagline, gradient effects, hexagonal icon
- **Minimum width**: 400px for optimal legibility

### 2. Icon Logo (`n3xus-vcos-icon.svg`)
- **Dimensions**: 200x200px (Square)
- **Usage**: App icons, favicons, social media avatars, profile pictures
- **Features**: Hexagonal design with "N3" text, multi-layered depth
- **Minimum size**: 64x64px

### 3. Horizontal Logo (`n3xus-vcos-horizontal.svg`)
- **Dimensions**: 600x100px
- **Usage**: Navigation bars, email signatures, compact spaces
- **Features**: Compact layout, icon + text inline

### 4. Vertical Logo (`n3xus-vcos-vertical.svg`)
- **Dimensions**: 300x400px
- **Usage**: Mobile applications, vertical banners, posters

## Clear Space
Maintain clear space around all logo variations equal to the height of one hexagon side from the icon. This ensures the logo has proper breathing room and maintains visual impact.

## Minimum Sizes
- **Primary Logo**: 400px width
- **Icon Logo**: 64x64px
- **Horizontal Logo**: 300px width
- **Vertical Logo**: 200px height

## Usage Guidelines

### DO ✓
- Use official logo files from the `/logos` directory
- Maintain aspect ratios when scaling
- Use on backgrounds with sufficient contrast
- Apply appropriate clear space
- Use SVG format when possible for best quality
- Use PNG format for raster requirements

### DON'T ✗
- Stretch or distort the logo
- Change logo colors outside the approved palette
- Add effects not present in official versions
- Place logo on busy backgrounds without backdrop
- Rotate the logo at odd angles
- Recreate or modify the logo design

## Background Usage

### Dark Backgrounds (Recommended)
The logo is optimized for dark backgrounds. Best visibility on:
- `#0a0e27` to `#1a1f3a` range
- Black (`#000000`)
- Dark gray (`#2a2a2a` and darker)

### Light Backgrounds
For light backgrounds, consider using a darker variant or adding a subtle dark backdrop/shadow behind the logo to maintain visibility of gradient effects.

## File Formats

### SVG (Scalable Vector Graphics)
- **Location**: `/logos/svg/`
- **Usage**: Web, print, any scalable application
- **Benefits**: Infinite scaling, small file size, editable

### PNG (Portable Network Graphics)
- **Location**: `/logos/png/`
- **Usage**: Raster applications, social media, presentations
- **Resolutions**: Available in multiple sizes (coming soon)

### ASCII Art
- **Location**: `/logos/ascii/`
- **Usage**: Terminal applications, CLI tools, console output, README files
- **Variants**: Large, medium, small, and banner formats

## Social Media Specifications

### Profile Pictures
- Use: `n3xus-vcos-icon.svg`
- Recommended size: 400x400px exported PNG

### Cover/Banner Images
- Use: `n3xus-vcos-primary.svg`
- Recommended sizes:
  - Twitter: 1500x500px
  - Facebook: 820x312px
  - LinkedIn: 1584x396px

### Post Graphics
- Incorporate brand colors
- Use hexagonal patterns when appropriate
- Maintain gradient aesthetic

## Print Guidelines
When printing the logo:
- Use high-resolution exports (300 DPI minimum)
- CMYK color conversion may slightly alter appearance
- Test print on target media before final production
- Ensure minimum sizes are respected

## Digital Guidelines
For digital displays:
- Use SVG format for responsive designs
- Implement proper fallbacks for older browsers
- Consider device pixel ratios for retina displays
- Optimize PNG exports for web (compress without quality loss)
